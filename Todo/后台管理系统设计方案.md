## 系统定位

### 网站特点
- **内容类型**：粉丝博客 + 资讯平台
- **核心内容**：文章、演出行程、视频、图片画廊
- **用户群体**：汪峰粉丝、音乐爱好者
- **运营模式**：管理员主导内容发布 + 用户互动

### 管理需求
1. **内容管理**：高效的内容发布和编辑流程
2. **审核机制**：内容质量把控
3. **数据分析**：了解用户行为和内容表现
4. **互动管理**：评论、留言、用户反馈
5. **SEO优化**：提升搜索引擎可见度

## 侧边栏导航设计

### 📐 设计原则
1. **功能分组清晰**：按业务逻辑分组
2. **层级扁平化**：最多2级菜单
3. **使用频率优先**：常用功能靠前

```

```

- 保存每次编辑的版本
- 版本对比
- 一键回滚

**3. 定时发布**
```javascript
{
  schedule_publish_at: "2024-01-20 18:00:00",
  auto_unpublish_at: "2024-02-20 23:59:59",  // 可选：自动下架

```

**4. 相关文章推荐**
- 自动根据标签/分类推荐
- 手动选择相关文章
- 推荐算法优化

**5. SEO优化工具**
```
- 自动生成SEO标题
- 关键词密度分析
- 元描述建议
- 结构化数据预览
- 社交媒体预览（Open Graph）
```

**6. 内容统计**
```
- 阅读量

- 完成率
- 跳出率
- 分享次数
- 收藏次数
```
